# Repo for OS course mini-project on </br>filesystems using FUSE for Google Drive  

## Usage  
\$ git clone https://github.com/akashdeeps19/gdrivefs.git  
\$ cd gdrivefs  
\$ pip install -r requirements.txt  
\$ ./mount.sh full/path/to/mountPoint
